# ArchSync

Monorepo scaffold:

- `frontend`: React + Vite (npm)
- `backend`: Python app (uv)

## Quick Start

```bash
# frontend
cd frontend
npm install
npm run dev
```

```bash
# backend
cd backend
uv sync
uv run python main.py
```
